Программа реализует LL(1) и LR(1) анализаторы для соответствующих КС-грамматик
На вход подавать текстовый файл с описанием грамматике в формате:
    -первая строка содержит нетерминалы (первый нетерминал распознается как начальный)
    -вторая строка терминалы
    -с третей строки правила Т->a b c (| - не допускается; символы разделять пробелами)
    -строка помеченная # - это комментарий к грамматике
    (Примеры можно найти в папке tests)

Далее программа попросит ввести строку для проверки.
Сначала запустится LL, потом LR. Оба анализатора построят вывод и ответят, принадлежит введенное выражение грамматике или нет.

Как установить программу:
1. Собрать с помощью ant
    зайти с консоли в директорию с программой запустить ant
    в этой же директории появится jar файл
    запустить командой java -jar Автоматы.jar
